Silent Network Authentication (SNA) at Signal 2022

I built a demo iOS application that showcases Silent Network Authentication (SNA) during Twilio's Developer conference at Signal 2022.

Client

Developers

Service

iOS, SNA, Security, Mobile Development, Authentication

Date

September 26, 2022

Challenge

How can you authenticate users without impacting the customer experience?

Solution

SNA verifies a phone number in the background without requiring user input.

OTP (One-Time Password) methods validate user authentication but the user experience is compromised. Users are sent a password which they then must enter to verify their identity. With SNA, the verification happens in the background for a seamless user experience.
